{{% blocks/lead color="primary" %}} ready4 provides tools for developing, retrieving, extending and applying the three core components of an open source youth mental health modelling project. {{% /blocks/lead %}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fa-lightbulb" title="Models" url = "docs/models/" %}} Models are abstract and generalised representations of systems described in computer code. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Data" url="docs/data/" %}} Data that describe a specific decision context context can be fed to models so they can be applied to that context. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-twitter" title="Analyses" url="docs/analyses/" %}} Analyses pair models with data, specifying which algorithms from the former are applied to the later and generating insights to inform potential policy or service design decisions. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}

Our aim is to develop and combine reference models of different systems that shape the mental health and wellbeing of young people.

{{< /blocks/section >}} {{< blocks/section >}} {{< cardpane >}} {{< card header="People" >}} Modelling the characteristics, relationships, behaviours, risk factors and outcomes of young people. {{< /card >}} {{< card header="Places" >}} Modelling demographic, environmental and proximity drivers of access, equity and outcomes. {{< /card >}} {{< /cardpane >}} {{< cardpane >}} {{< card header="Platforms" >}} Modelling the optimal staffing and configuration of support services for young people. {{< /card >}} {{< card header="Programs" >}} Modelling the affordability, value for money and appropriate targeting of interventions for young people. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/section >}}
