{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} readyforwhatsnext is an open source project to develop and combine reference models of systems that shape the mental health and wellbeing of young people. {{% /blocks/lead %}} {{< blocks/section >}} {{< cardpane >}} {{< card header="Models" >}} Each self-contained modelling project is a module. Modules can be combined to model more extensive systems. {{< /card >}} {{< card header="Data" >}} Models are fed data, specific to the decision context. All non-confidential data are publicly available. {{< /card >}} {{< card header="Analysis" >}} When models are supplied with data, they generate insight to help improve the lives of young people. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/section >}}

readyforwhatsnext aims to develop and combine reference models on multiple types of systems..

{{< /blocks/section >}} {{< blocks/section >}} {{< cardpane >}} {{< card header="People" >}} Modelling the characteristics, relationships, behaviours, risk factors and outcomes of children and young people. {{< /card >}} {{< card header="Places" >}} Modelling the role of area and proximity characteristics on outcomes, exploring geographic equity. {{< /card >}} {{< /cardpane >}} {{< cardpane >}} {{< card header="Platforms" >}} Modelling the accessibility, efficient configuration and resourcing requirements of health services for young people. {{< /card >}} {{< card header="Programs" >}} Modelling the affordability, value for money and appropriate targeting of interventions to improve mental health and wellbeing outcomes in young people. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/section >}}

As an open source project, we welcome your contributions!
