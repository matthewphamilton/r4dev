{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} ready4 is an open source project to develop and combine reference models of systems shaping the mental health and wellbeing of young people. {{% /blocks/lead %}} {{< blocks/section >}} {{% blocks/feature icon="fab fa-app-store-ios" title="People" %}} Modelling the characteristics, relationships, behaviours, risk factors and outcomes of young people. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Places" url="https://github.com/google/docsy-example" %}} Modelling demographic, environmental and proximity drivers of access, equity and outcomes. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-twitter" title="Platforms" url="https://twitter.com/mph_economist" %}} Modelling the optimal staffing and configuration of support services for young people. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-twitter" title="Programs" url="https://twitter.com/mph_economist" %}} Modelling the affordability, value for money and appropriate targeting of interventions for young people. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}

We provide three categories of open source tools.

{{< /blocks/section >}} {{< blocks/section >}} {{< cardpane >}} {{< card header="Models" >}} Models are abstract and generalised representations of systems described in "modules" of computer code. Modules can be combined to model more extensive systems. {{< /card >}} {{< card header="Data" >}} Models can be applied to a specific decision context by feeding them data that describe that context. {{< /card >}} {{< card header="Analyses" >}} Each analaysis applies a model to a dataset, generating insight that can be used to help improve the lives of young people. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/section >}}

As an open source project, we welcome your contributions!
